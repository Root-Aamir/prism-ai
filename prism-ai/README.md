## Overview

Prism is a Laravel package designed to simplify the integration of **Large Language Models (LLMs)** into modern web applications.

It provides a clean and flexible interface that allows developers to interact with AI services without dealing with complicated API implementations.

Using Prism, you can easily build **AI-powered Laravel applications** with features like automated text generation and conversational workflows.

---

## Key Capabilities

- Simple integration with Laravel projects  
- AI text generation support  
- Multi-step conversational flows  
- Tool and provider integrations  
- Clean and developer-friendly syntax
